# React + Django Chatbot (Full Project)

This archive contains a complete React frontend and Django backend scaffold for a rule-based chatbot.
Follow the README in the backend/frontend sections to install dependencies and run locally.

Backend:
  - cd backend
  - python -m venv venv
  - activate venv
  - pip install -r requirements.txt
  - python manage.py migrate
  - python manage.py createsuperuser (optional)
  - python manage.py runserver

Frontend:
  - cd frontend
  - npm install
  - npm start

After both servers run, open http://localhost:3000
